# Controller Manager
## Maybe Controller Addon would've been better

### An addon for the [AppLauncher](https://github.com/Tommy1250/AppLauncherUI) Made To allow the app to use native methods with the controller apis

Currently the addon only has the disconnectBT function whitch disconnects the controller fom Bluetooth provided the MAC adress

# To install
```bash
npm i github:Tommy1250/ControllerManager
```
